# Project: Sebooth Website (Web Portal)

## Purpose
A web portal designed for Sebooth photobooth users to access, view, and download their digital assets (softfiles) via QR code redirection from the physical photobooth stations.

## Technical Stack
- **Framework**: Next.js 16 (App Router)
- **UI/Logic**: React 19
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Language**: TypeScript

## Directory Structure
- `src/app/`: App Router pages and layouts.
- `src/components/`: Reusable UI components and page sections.
- `src/lib/`: Utility functions and shared library code.
- `public/`: Static assets (images, icons).

## Key Features (To be implemented)
1. **QR Access Point**: Handling dynamic routes for photobooth session IDs.
2. **User Authentication**: Account creation and login to manage personal galleries.
3. **Gallery Management**: Viewing photos, GIFs, and videos associated with a session.
4. **Softfile Downloads**: Secure download links for high-quality assets.
5. **Session Linking**: Associating anonymous photobooth sessions with registered user accounts.

## Coding Conventions
- Use Functional Components with TypeScript.
- Follow App Router paradigms (Server Components by default).
- Utility-first CSS with Tailwind 4.
- Modular component design.
